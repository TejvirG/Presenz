# Presenz Frontend

This folder contains the Flutter frontend for Presenz.

## Setup

1. Install Flutter SDK and required tools.
2. Open a terminal in `frontend/`.
3. Run:
   ```bash
   flutter pub get
   flutter run
   ```

## Notes

- The frontend uses `frontend/lib/utils/api_config.dart` to configure the backend API base URL.
- If you run the backend locally, update the URL there.
- Use Android/iOS emulators or desktop targets supported by Flutter.
